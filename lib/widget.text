14. Icon Selection Logic:
_getLocationIcon() returns:
├── Icons.terrain (ภูเขา)
├── Icons.waves (ทะเล)  
├── Icons.water (น้ำตก)
├── Icons.temple_buddhist (วัด)
├── Icons.store (ตลาด)
└── Icons.place (default)

15. Score Calculation Widget Flow:
TripLocation.calculateScore()
├── difficultyScore: (10 - difficulty) * 1.5
├── distanceScore: distance <= 50 ? 8 : (distance <= 100 ? 5 : 2)
└── unknownBonus: isUnknown ? 2 : 0

16. Rating Level Widget Flow:
TripLocation.getRatingLevel()
├── score >= 20: '🌟 ยอดเยี่ยม'
├── score >= 15: '⭐ ดีมาก'  
├── score >= 10: '👍 ดี'
└── default: '📍 พอใช้'

└── Form validation triggers
